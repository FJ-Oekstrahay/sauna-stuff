# Sauna Stuff 🔥

A comprehensive, evidence-based guide to sauna best practices.

## Contents

- **`index.html`** — Professional interactive HTML report with table of contents and inline CSS styling
- **`sauna-guide.md`** — Complete Markdown version of the guide
- **`README.md`** — This file

## Topics Covered

1. **Types of Saunas** — Finnish, infrared, steam, and hybrid saunas with pros/cons
2. **Temperature & Time Guidelines** — Evidence-based recommendations by sauna type
3. **Health Benefits** — Cardiovascular, sleep, immune, stress, athletic, skin, and metabolic benefits with citations
4. **Best Practices** — Pre-, during, and post-sauna protocols
5. **Contraindications** — When NOT to use a sauna
6. **Myths vs. Reality** — Common misconceptions debunked

## Research Sources

- Finnish cohort studies
- JAMA Internal Medicine (2015)
- Meta-analyses on heat therapy and wellness outcomes
